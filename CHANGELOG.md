## [0.1.0] - 2025-10-07

### Added
- Initial release of API Debugger
- Core APIClient with debugging capabilities
- Support for both requests and httpx backends
- Beautiful Rich-powered console logging
- Data masking for sensitive fields
- cURL command generation
- Retry mechanism with configurable delays
- Django REST Framework middleware integration
- FastAPI middleware integration
- Comprehensive configuration system
- Environment variable support
- File and console logging options
- Complete test suite with 90% pass rate
- Comprehensive documentation and examples

### Features
- 🚀 Standalone API client with debug logging
- 🔧 Framework integrations (Django DRF, FastAPI)
- 📊 Rich logging with pretty formatting
- 🎨 Syntax-highlighted JSON output
- 🔒 Automatic sensitive data masking
- ⚙️ Flexible configuration options
- 📋 cURL export functionality
- 🔄 Smart retry logic
- ⏱️ Request/response timing
- 📝 Comprehensive test coverage
